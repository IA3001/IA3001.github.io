算法竞赛入门指南

· 2026-07-21 · 1.1 千字 ·

前言

算法竞赛是锻炼编程能力和逻辑思维的最佳方式之一。无论你是准备参加 ACM-ICPC、Codeforces 还是蓝桥杯，掌握基础算法都是第一步。

数据结构基础

数组与链表

数组是最基本的数据结构，支持 O(1) 的随机访问。但在中间插入和删除元素时需要 O(n) 的时间。

链表则相反，插入和删除只需 O(1)，但访问第 k 个元素需要 O(k)。

栈与队列

栈是后进先出（LIFO）的数据结构，常用于括号匹配、表达式求值、DFS 等场景。

队列是先进先出（FIFO）的结构，BFS 中的核心数据结构就是队列。

单调栈

单调栈在 O(n) 时间内解决”下一个更大元素”类问题，是竞赛中的高频考点。

单调队列

滑动窗口最大值是单调队列的经典应用。

树与图

树是一种特殊的图，没有环。二叉树、BST、堆、并查集都是树的应用。

图由顶点和边组成，分为有向图和无向图，存储方式有邻接矩阵和邻接表。

基础算法

排序算法

快速排序

平均时间复杂度 O(n log n)，基于分治思想。选择一个 pivot，将数组分为两部分递归排序。

归并排序

稳定排序，时间复杂度 O(n log n)，同样基于分治。归并排序的另一大用途是求逆序对。

二分查找

在有序序列中查找目标值，每次将搜索范围减半，时间复杂度 O(log n)。二分答案也是常见题型。

前缀和与差分

一维/二维前缀和可以在 O(1) 时间内求区间和。差分数组则用于频繁的区间修改操作。

动态规划

线性 DP

最长上升子序列 (LIS)

经典 DP 问题，朴素 O(n²)，贪心+二分优化到 O(n log n)。

背包问题

01 背包、完全背包、多重背包，是 DP 中最经典的模型。

区间 DP

从小区间推到大区间，常见于合并石子、矩阵链乘等问题。

树形 DP

在树上进行 DP，通常用 DFS 实现。典型问题：树的直径、树上最大独立集。

图论算法

最短路

Dijkstra 算法

适用于非负权图，使用优先队列优化后复杂度 O((V+E) log V)。

Floyd-Warshall

多源最短路，O(n³)，适用于稠密图或需要全部点对距离的场景。

最小生成树

Kruskal 算法

基于并查集，按边权排序后贪心选取。时间复杂度 O(E log E)。

拓扑排序

用于 DAG，可以判断图中是否有环，也常用于 DP 的状态转移顺序。

字符串算法

字符串哈希

将字符串映射为整数，快速判断子串是否相等。常用的有自然溢出和双哈希。

KMP 算法

单模式串匹配，通过前缀函数避免重复比较，时间复杂度 O(n+m)。

数学基础

数论

快速幂、GCD/LCM、素数筛（埃氏筛、欧拉筛）、乘法逆元都是必备技能。

组合数学

排列组合、容斥原理、卡特兰数、Lucas 定理在竞赛中频繁出现。

结语

算法学习没有捷径，多刷题、多总结、多看题解是最好的方法。祝你在竞赛中取得好成绩！
