1006 合成大hdu

Problem Description

河灵是 hdu 的狂热粉丝。在他眼中，一切的一切都是 hdu 的模样。

有一天晚上，河灵在草稿纸上涂鸦时发现，他居然可以在一个字符串中窥见 hdu 的影子！对于一个字符串 SSS，河灵可以一下子就数出字符串 SSS 的不同子序列 hdu 的个数。

简单的数数已经不能满足河灵了。河灵有一个正整数 nnn (1≤n≤1091 \le n \le 10^91≤n≤109)，河灵很想知道拥有不同子序列 hdu 个数恰好为 nnn 的字符串 SSS 是什么样的。

请你帮帮河灵，你需要构造一个仅包含 h, d, u 三种字符的字符串 SSS，使得字符串 SSS 的不同子序列 hdu 的个数恰好等于 nnn。但是河灵的草稿纸大小有限，所以你构造的字符串长度不能超过 300130013001。

可以证明，对于所有满足 1≤n≤1091 \le n \le 10^91≤n≤109 的正整数 nnn，至少存在一种满足要求的构造方案。

注：

  • 子序列：如果 S′S'S′ 可以通过 SSS 删除若干个（可能是零个或全部）元素，且不改变剩余元素的相对顺序得到，则称 S′S'S′ 是 SSS 的子序列。
  • 不同的子序列：两个子序列 S1,S2S_1, S_2S1​,S2​ 不同，当且仅当原序列中至少存在一个位置在一个子序列中出现，在另一个子序列中被删除。

Input

每个测试点中包含多组测试数据。输入的第一行包含一个正整数 TTT (1≤T≤1031 \le T \le 10^31≤T≤103)，表示数据组数。对于每组测试数据：

一行一个正整数 nnn (1≤n≤1091 \le n \le 10^91≤n≤109)，表示构造的字符串 SSS 需要包含的不同子序列 hdu 的个数。

Output

对于每组测试数据：输出一行一个字符串 SSS (1≤∣S∣≤30011 \le |S| \le 30011≤∣S∣≤3001)，表示你构造的字符串。你需要保证字符串 SSS 仅包含 h, d, u 三种字符。

若存在多种满足条件的字符串 SSS，输出任意一种即可。

Sample Input

3
1
3
27

Sample Output

hdhdu
hhhddduuu

#include<bits/stdc++.h>
using namespace std;

bool check(string s, int ans) {
    int a = 0, b = 0, c = 0;
    for(auto _: s) {
        if (_ == 'h') a++;
        else if (_ == 'd') b += a;
        else c += b;
    }
    // cout << a << " " << b << " " << c << "\n";
    return c == ans && s.size() <= 3001;
}

string cal1(int n) {
	// n <= 1e6
	int B = 1500;
	int q = n / B, r = n % B;
	return string(r, 'h') + "d" + string(B - r, 'h') + string(q, 'd') + "u";
}

string cal2(int n) {
	// n > 1e6
	int B = 1000, C = 999;
	int r = B - n % B, m = (n - C * r) / B, q = m / C, s = m % C;
    // ??? 神秘特判，要不然 999999999 的时候 q = 1001, s = 0
    if (!s && q) q--, s+=C; 
	return string(r, 'h') + "d" + string(B - r, 'h') + string(q, 'd') + string(C - s, 'u') + "d" + string(s, 'u');
}

void solve() {
	int n;
	cin >> n;
    string res = (n <= 1e6 + 5? cal1(n) : cal2(n));
    // if (!check(res, n)) {
        // cout << n << ":";
        // cout << res.size() << "\n";
    // }
    cout << res << "\n";
}
int main() {
	ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
	int T = 1;
	cin >> T;
	while (T--) solve();
}
